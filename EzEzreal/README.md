## EzEzreal
**Features**:
- Combo:
  - Q/W Combo.
  - You can choose the option that the W will use only if the Q can hit.
- Harass:
  - Q Harass.
  - Mana Manager.
- Lane Clear:
  - Q LaneClear.
  - Use Q only for LastHit.
  - Mana Manager.
- Jungle Clear:
  - Q Jungle Clear.
  - Mana Manager.
- Kill Steal:
  - Q KS.
  - R KS.
- Miscellaneous:
  - Auto Q to harass.
  - Semi R to target (T default key).
  - Mana Manager.
- Draw:
  - Draw Q, W, E range.
